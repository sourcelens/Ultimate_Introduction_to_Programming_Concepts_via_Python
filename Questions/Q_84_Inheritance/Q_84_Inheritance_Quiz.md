1 : The concept in Object oriented programming where we try to reuse the code from a parent class to child class or derived class is called _______  

a) Polymorphism  
b) Inheritance   
c) Encapsulation  
d) All of the above  

**Answer** b) 

**Description**  

The concept in Object oriented programming where we try to reuse the code from a parent class to child class or derived class is called Inheritance.

2 : Inheritance in Object oriented programming is also called as _______  

a) Is a relation   
b) Is not a relation   
c) Any of the above   
d) None of the above  

**Answer** a) 

**Description**

Inheritance in Object oriented programming is also called as ‘Is a relation’.
