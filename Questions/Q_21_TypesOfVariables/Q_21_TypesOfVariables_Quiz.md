1 : Based on purpose variables are divided into _______  

a) Float & int  
b) 8 bit, 16 bit, etc.  
c) Signed & unsigned  
d) All of the above  

**Answer** a) 

**Description**  

Float is normally used to store decimals. Eg:- 1.25, 2.5, etc. Int is used for integers like 1, 2, 3, etc.
